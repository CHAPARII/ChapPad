# ChapPad
I've madee this hackpad by my experience which I've gained while working on my keyboard :b. Also by making this I'll add this hackpad in my setup with the keyboard!! 

This hackpad have 11keys and 1 rotary encoder, I've used XIAO RP2040 microcontroller in this and a 0.9'' OLED Display. I've done the BOM and made it from one supplier "Amazon" to make clean or else it could become messy. I've some keys in my home as well (1-2 keys) which I'll use in if it's needed. :b For the CAD I made it in two parts "top" and "bottom". I also thought to make a top plate but that would look over artificial and I wanted my Hackpad to look like "YES!! This is coustom hackpad, which I made by myself". I'll order the PCB in red or blue color because will match with my setup T_T.. wuuhuuuu I'm so exited to build this!!!

The CAD designe is looking preety simple (as I made it intentionally), but the PCB became so rocking cool!!

Till now everything is good to go! yay!
